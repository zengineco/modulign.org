The Grammar

What a Modulign
Address Encodes

A minimum valid address — five required segments — is immediately useful. Every additional segment adds precision without invalidating the base code.

DOM · SUB
Domain
What type of observable environment. Nine domains: URB NAT TRN COM RES EVT SEC INF UNK.
REALM / G1–G4
Locus
Where in physical space. Five realms (SR AT AQ OR VR) × four-level geo hierarchy. Monotonic: no city without country.
NOD
Node
The specific instrument or site. Base36⁴ = 1,679,616 unique nodes per locus.
|SCL|
Scale
At what scale observed. Seven tiers: |QNT| |MLC| |BIO| |HMN| |GEO| |STL| |CSM|. Wrong scale = false classification.
:MED
Medium
How captured. Eleven codes including :SYN — the synthetic content primitive. VR/ + :SYN ↔ SyntheticContent = TRUE.
%OBS
Observer
Who classified and at what competence. %HUM %AUT %HYB × domain × :T :E :A. Gates ^EVID eligibility.
@TIME
Timestamp
ISO 8601. Required for ^EVID grade. Immutable in the append-only Observational Ledger.
§JUR
Jurisdiction
Governing legal framework. Constitutive in ^EVID classifications. Encodes which admissibility standard applies.
^META
Evidence Grade
^EVID: append-only, cryptographically signed, satisfies FRE 901(b)(9). ^PROV: provisional, excluded from proceedings.

Formal Results

What the Architecture
Formally Proves

These results are consequences of the architecture, not motivations for it. The Standard was built to classify television channels. The epistemology was an accidental discovery during formalization.

Theorem 4.1 · Formal Logic v1.1.0
ATNA
∀ Stepₙ ∈ CDP : justificatory_input(Stepₙ) ⊆ F(ω)
T(C) → F(ω) → J(C,ω) → C
∴ C is non-accidentally true when true. □
The Address-Theoretic Non-Accidentality Principle. A classification produced by the full Decision Protocol cannot be accidentally true. The structural precondition for Gettier cases — logical independence of justification from truth — does not exist in the architecture.
Theorem 4.1C · Formal Logic v1.1.0
Fake Barns Dissolved
scale(T(C)) = |GEO| (regional)
SCL(Henry) = |HMN| (perceptual)
SCL(C) ≠ scale(T(C)) → C is FALSE. □
Goldman's fake barns case — the hardest Gettier case, which defeats every fourth-condition proposal — is dissolved by scale-sensitivity. The architecture correctly refuses to certify barn-knowledge at the wrong scale, not by adding a condition to JTB, but by identifying the scale error that JTB cannot encode.
Invariant I6 · Standard v3.0
Synthetic Content Primitive
REALM = VR ∧ MED = SYN
↔ SyntheticContent(ω) = TRUE
Not a tag. A structural biconditional.
VR/·:SYN is the first formal epistemic primitive for synthetic content. Synthetic content has no physical locus and cannot be assigned a physical realm — a consequence of Definition 1.1(a). The biconditional is architecturally enforced, not policy-dependent.
Theorem 6.1 · Formal Logic v1.1.0
Evidentiary Completeness
∀e ∈ {Auth, CoC, BestEvidence, Daubert}
satisfied(e, C^EVID) = TRUE. □
FRE 901(b)(9) · FRE 1001–1008 · FRE 702
Every ^EVID-grade classification satisfies all four foundational legal admissibility requirements as constitutive outputs of the CDP — not as post-hoc documentation. The error rate is structurally derivable: 1 − κ per classification.
Corollary 4.1 · Formal Logic v1.1.0
Zagzebski's Recipe Fails
Recipe entry point: w' where X satisfied, T(C) fails.
Under ATK: X = CDP requires F(ω).
If T(C) fails, F(ω) severs. CDP cannot complete. □
Zagzebski (1994) proved no analysis K = df true belief + X escapes Gettier cases as long as X is fallible. ATK escapes her scope condition by abandoning the K = df true belief + X form entirely — replacing it with a constitutive procedural architecture.
Theorem 4.2 · Formal Logic v1.1.0
Intersubjectivity
CDP(ω,O₁) = CDP(ω,O₂)
for O₁,O₂ with equivalent competence
modulo threshold boundaries. □
Any two competent observers applying the CDP to the same observable features must converge on the same classification. Divergence at threshold boundaries is a first-class data point — correctly encoded as lower confidence — not an error.
